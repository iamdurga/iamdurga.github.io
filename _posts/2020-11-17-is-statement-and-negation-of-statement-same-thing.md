 Write converse inverse and contrapositive of the statement.
    “ if x < 0 then $$ x^2 – x > 0 $$ “, Also write the negation of each statement.
    
    
Solution:

Given statement is 

    $$ " if x < 0   then ,   x^2 – x > 0 " $$ 

 Converse:

$$ if x^2 – x > 0  then ,  x < 0.$$ 

 Nagation:

$$ x^2 – x > 0  and ,  x > 0 $$ 

 Inverse 
		 
$$ if x > 0  then ,  x^2 – x < 0.$$

 Negation:
		 
$$ x > 0  and ,  x^2 – x > 0. $$ 

Contrapositive:
		  
$$ if  x^2 – x  < 0  then ,  x > 0 $$

 Negation:
		
$$ x^2 – x  < 0  and ,  x < 0. $$
